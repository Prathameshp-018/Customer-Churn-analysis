# Customer Churn Analysis 📊

## 📌 Project Overview
This project analyzes customer churn behavior using the Telco Customer Churn dataset.  
The goal is to identify key factors that influence customer retention and churn.

---

## 🎯 Objective
- Analyze customer data
- Identify patterns behind customer churn
- Provide insights to improve retention

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
- Telco Customer Churn Dataset
- Contains customer details like tenure, services, contract type, and churn status

---

## 🔍 Data Preprocessing
- Handled missing values
- Removed duplicate records
- Converted categorical values (e.g., 0 → No, 1 → Yes)

---

## 📊 Exploratory Data Analysis (EDA)
Performed analysis using:
- Count plots
- Bar charts
- Pie charts

---

## 📈 Key Insights
- Around **26.54% customers have churned**
- Customers with **low tenure** are more likely to churn
- **Senior citizens** show higher churn rate
- **Month-to-month contract** users churn more
- Customers without **online security & tech support** churn more
- **Fiber optic users** have higher churn
- **Electronic check payment method** has highest churn
- Customers using **multiple services** are less likely to churn

---

## 📌 Conclusion
Customer churn is mainly affected by:
- Contract type
- Service usage
- Customer engagement

Improving service quality and encouraging long-term contracts can reduce churn.
